Notatki z WTF: Co ten frontend

Życie w pudełku - jednostki

  • 0 - zero to zawsze zero ;) nie musimy pisać jednostki.
  • Piksele - nasz ekran składa się z pikseli, dlatego jest to podstawowa miara. Skrót to px.
  • Procenty - jest to zawsze procent z wielkości parenta
  • vw - viewport weight - procentowo, ale nie z parenta, a z całego okna.
  • vh - viewport height - procentowo, ale nie z parenta, a z całego okna.

Życie w pudełku: Box model (padding / border / margin)

Padding - wewnętrzny margines

Sposoby zapisu
  • Padding-top, Padding-right, Padding-bottom, Padding-left
  • Padding: 30px 40px 50px 60px - góra, prawo, dół, lewy
  • Padding: 30px 40px - padding lustrzany, pierwsza określa dół i górę, a druga lewy i prawy

Życie w pudełku: float

float - powoduje, że element jest wcięty w tekst, opływany przez resztę elementów

clear - powoduje, że element rozszera się wraz z float

JS: Typy danych i immutability

Najważniejsze fakty:

  • const - stała zmienna, nie możemy jej nadpisać
  • let - zmienna, pozwala zmieniać przypisanie
  • W JS używamy camel case, czyli newName.
  • typeof - funkcja, która pozwala sprawdzić typ zmiennej

Typy proste w JS

  • String - tekst, otaczamy ' lub " (zalecam ')
  • Number - liczba
  • Bigint - duża liczba ;) powyżej 2^53
  • boolean - true lub false
  • null - wartość celowo niezdefiniowana
  • undefined - brak definicji
  • *symbol - tworzy "unikalne opakowanie"

Jak pobrać element HTML przez JS?

document.querySelector('selector')

Podsumowanie:

  • DOM to "złożony" w przeglądarce HTML
  • JavaScript może modyfikować DOM
  • funkcja querySelector zwraca node (węzeł) z HTML-a selektor jest zapisywany zgodnie z css
