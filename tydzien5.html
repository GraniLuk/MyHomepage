HTML: Listy, tabele, definicje

  • ul - unordered list
  • ol - ordered list
  • dl - description list
  • dd - description details
  • Zagnieżdżanie list
  • table - description details
    • tr - wiersz tabeli (table row)
    • tr - komórka tabeli (table data)
    • thead - nagłówek tabeli
    • th - komórka nagłówka tabeli
    • tbody - ciało tabeli
    • caption - opis tabeli
