Porównania

  • < - lewa strona większa od prawej
  • > - prawa strona większa od lewej
  • ≤ - lewa strona większa lub równa od prawej
  • ≥ - prawa strona większa lub równa od lewej
  • == - obie strony równe
  • == - obie strony równe, sprawdza typeof
  • == - obie strony różne
  • == - obie strony różne (sprawdza typeof)

Formularze: GET i POST

Do znacznika form, możemy dodać parametry:

  • action - Pod jaki adres będzie wysłany formularz.

Atrybuty walidacji:

  • Required - wymagane pole
  • pattern - Regexy ;)
Walidacja po stronie frontendu nie zastępuje walidacji po stronie backendu!

CSS: Grid

Własności gridowe:

  • grid-template-columns: - definicja kolumn (wartości podajemy po spacji)
  • grid-template-rows: - definicja wierszy (wartości podajemy po spacji)
  • fr: - nowa definicja miary, procentowy rozkład miejsca, bierze pod uwagę tylko wolne miejsce, czyli możemy mieszać jednostki
  • repeat: - zamiast definiować każdą kolumnę osobno, możemy użyć repeat.
  • repeat: - zamiast definiować każdą kolumnę osobno, możemy użyć repeat.
  • column-gap: - odstępy między kolumnami.
  • row-gap: - odstępy między wierszami.
  • minmax: - możemy zdefiniować minimalną i maksymalną wartość np. szerokości kolumny.

CSS: Grid – podstawowy layout

Nowe znaczniki:

  • grid-template-areas - szablon obszarów na stronie
  • grid-column-start - można specyfikować od której kolumny dany element się zaczyna
  • grid-column-end - można specyfikować na której kolumnie dany element się kończy
